ScriptText

a superfast, small and minimalistic text editor for the UWP.

my try to get into UWP and C#

ScriptText is an super fast, simple and minimalistic texteditor while looking beautifully and lightweight. Made as simple as possible. With bigger font-size, reduced buttons and an smaller text-width it should be best editor to read and write .txt-files.
For better focus.

Written in C# the code is visible at GitHub.
