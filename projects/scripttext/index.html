About

ScriptText is an super fast, simple and minimalistic texteditor while looking beautifully and lightweight. Made as an awesome writing environment. With bigger font-size, reduced buttons and an smaller text-width it should be best editor to read and write .txt-files. Minimalistic look for better focus.

Written in C# the code is visible at GitHub.
